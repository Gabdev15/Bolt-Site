Quand tu modifies le code :

1. Crée une branche avec ce format :
   feature/<description-courte>

2. Commit avec message clair :
   feat: <description>

3. Push la branche

4. Crée une pull request vers main avec :
   - titre clair
   - description des changements
   - pourquoi c’est utile
