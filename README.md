## 📋 About

**Bolt-Site** is the official website for the **Bolt** roleplay enterprise on a Garry's Mod RP server. Inspired by the real [Bolt.eu](https://bolt.eu) company, this site serves as the public face of the in-game corporation — presenting the company, its services, and its identity within the roleplay universe.

---

## 🚀 Features

- 🏠 **Landing page** — Company presentation & branding
- 🚗 **Services** — Overview of Bolt's (RP) activities
- 🎨 **UI** — Design inspired by the real Bolt brand
- 📱 **Responsive** — Works on all screen sizes

## ⚙️ Installation

```bash
# Clone the repo
git clone https://github.com/GabVS15/Bolt-Site.git

# Go into the project
cd Bolt-Site

# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## 🌍 Context

> This project is part of a **Garry's Mod roleplay** experience where players run fictional enterprises inspired by real companies. Bolt is one of these enterprises — operating as a VTC/mobility company within the RP universe.
